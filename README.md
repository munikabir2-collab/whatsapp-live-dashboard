# WhatsApp Live Dashboard

WhatsApp Live Dashboard is a real-time web-based system to monitor and manage WhatsApp messages, chats, and user activity from a single admin panel.

---

## 🚀 Features
- Real-time message tracking  
- Live chat monitoring  
- User activity status  
- Message delivery reports  
- Admin dashboard interface  
- Fast and responsive UI  

---

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript (or React if used)
- Backend: Node.js / Python (update as per your project)
- Database: MongoDB / MySQL (if used)
